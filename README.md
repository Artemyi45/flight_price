# Flight price prediction
Бот, помогающий купить самые выгодные авиабилеты

## Проблема
У покупателя есть информация о цене авиабилета сейчас, но нет о цене в будущем

## Решение
Рекомендация к покупке в наиболее оптимальную-предсказанную дату

## Как это работает
- **Данные:** парсинг с разных источников
- **Модель:** в процессе обучения...
- **Результат:** несколько вариантов ответа. Ориентировочная дата, точная дата.

## Текущий статус

Пробное обучение модели
Первые выводы
